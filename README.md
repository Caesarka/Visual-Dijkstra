An interactive web application that visualizes Dijkstra's shortest path algorithm on a map of connected cities. The project demonstrates how the algorithm finds the shortest path step by step using dynamically calculated edge weights. Users can select the start and destination cities.

Features:
- Interactive SVG map
- Select start and destination cities
- Step-by-step visualization of Dijkstra's algorithm
- Automatic edge weight calculation from city coordinates
- Shortest path reconstruction

Technologies:
- HTML5
- CSS3
- JavaScript (ES6)
- SVG

How to Run:
- Clone the repository
- Open index.html in your browser.

Live Demo: https://visual-dijkstra.gusarov.com/
